For anything missing or unclear, check `resources/tidb-terms.md` (the TiDB 中英术语表) before making a translation choice. That file is the source of truth; this table is only a fast lookup for high-frequency terms.

## Usage notes

- Keep product and feature names in English when that is the established repo usage.
- Keep commands, flags, file paths, and configuration keys unchanged.
- Add a space between Chinese characters and adjacent English words, numbers, or inline code.
- Prefer consistency with the target repo over ad hoc wording improvements.
