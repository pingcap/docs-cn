# TiEM 1.0.1 Release Notes

2022 年 5 月 30 日，TiEM 发布 1.0.1 版。该版本在 1.0.0 版的基础上，对 TiEM 安全性做出了改进，并增加了 TiDB 支持的版本。

## TiEM

- 新增支持 TiDB 6.0 的部署
- 优化 workflow 逻辑，支持重启继续执行
- 修复一些安全基线漏洞
- 替换 Elastic Search, Filebeat, Kibana, Grafana 为 OSS 版本
