# SHOW BUILTINS

`SHOW BUILTINS` 语句用于列出 TiDB 中所有的内置函数。

## 语法图

**ShowBuiltinsStmt:**

![ShowBuiltinsStmt](/media/sqlgram/ShowBuiltinsStmt.png)

SHOW BUILTINS;

## MySQL 兼容性

`SHOW BUILTINS` 是 TiDB 新增语法，MySQL 中无此语法。
