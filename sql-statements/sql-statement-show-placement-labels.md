# SHOW PLACEMENT LABELS

`SHOW PLACEMENT LABELS` 汇总可用于放置规则 (Placement Rules) 的标签 (label) 和值。

## 语法图

```ebnf+diagram
ShowStmt ::=
    "PLACEMENT" "LABELS"
```

## 示例

{{< copyable "sql" >}}

```sql
SHOW PLACEMENT LABELS;
```

```sql
+--------+----------------+
| Key    | Values         |
+--------+----------------+
| region | ["us-east-1"]  |
| zone   | ["us-east-1a"] |
+--------+----------------+
2 rows in set (0.00 sec)
```

## MySQL 兼容性

该语句是 TiDB 对 MySQL 语法的扩展。
