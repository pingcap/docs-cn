# SHOW [FULL] FIELDS FROM

`SHOW [FULL] FIELDS FROM` 是 [`SHOW [FULL] COLUMNS FROM`](/sql-statements/sql-statement-show-columns-from.md) 的别名。包含该语句提供了 MySQL 兼容性。
