# SHOW STATS_HISTOGRAMS

你可以使用 `SHOW STATS_HISTOGRAMS` 语句查看统计信息中直方图的相关信息。

![ShowTargetFilterable](/media/sqlgram/ShowTargetFilterable.png)

```sql
show stats_histograms;
```

```sql
show stats_histograms where table_name = 't2';
```
