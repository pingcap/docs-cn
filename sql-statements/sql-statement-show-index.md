# SHOW INDEX [FROM|IN]

`SHOW INDEX [FROM|IN]` 语句是 [`SHOW INDEXES [FROM|IN]`](/sql-statements/sql-statement-show-indexes.md) 的别名。包含该语句提供了 MySQL 兼容性。
