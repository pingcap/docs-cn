# SHOW KEYS [FROM|IN]

`SHOW KEYS [FROM|IN]` 语句是 [`SHOW INDEXES [FROM|IN]`](/sql-statements/sql-statement-show-indexes.md) 语句的别名。包含该语句提供了 MySQL 兼容性。
