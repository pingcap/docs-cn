# TiDB 数据操作语言

数据操作语言（Data Manipulation Language， DML）用于帮助用户实现对数据库的基本操作，比如查询、写入、删除和修改数据库中的数据。

TiDB 支持的数据操作语言包括 Select ，Insert, Delete, Update，和 Replace。
