# TiFlash 命令行参数

本文介绍了 TiFlash 的命令行启动参数。

## `server --config-file`

+ 指定 TiFlash 的配置文件路径
+ 默认：""
+ 必须指定配置文件，详细的配置项请参阅 [TiFlash 配置参数](/tiflash/tiflash-configuration.md)
